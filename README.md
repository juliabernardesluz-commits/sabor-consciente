# sabor-consciente

criei esse site porque percebi que muita gente na nossa escola acha que sustentabilidade e alimentação saudável são assuntos chatos ou só para adultos. A minha ideia com o Sabor Consciente foi usar a tecnologia para mostrar que a gente, como estudante, pode mudar as coisas começando pelo nosso próprio prato.

Eu escolhi esses tons de verde e esse fundo mais claro para lembrar a natureza e a horta, deixando o visual bem limpo e direto. O texto foi escrito do meu jeito, com uma linguagem simples, para que qualquer colega que acessar entenda o recado logo de cara: valorizar o produtor local, comer mais comida de verdade e não desperdiçar o que sobra.

Na parte técnica, eu me desafiei a fazer um layout responsivo. Usei o conceito de Grid no CSS para que as informações fiquem bem organizadas em colunas no computador, mas se adaptem sozinhas, virando uma lista, se alguém abrir o site pelo celular no meio do recreio.

No fim, esse projeto é a minha forma de mostrar que a tecnologia pode ser uma ferramenta incrível para espalhar as ideias do Agrinho e fazer a nossa parte por um futuro melhor!
